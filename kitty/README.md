## Customized Keyboard Shortcuts
### Window Panes
- New pane
    - `super + return (enter)`
- Next Pane
    - `super + shift + right`
- Previous Pane
    - `super + shift + left`
- Next Pane Layout
    - `super + shift + l`

### Tabs
- New Tab
    - `super + t`
- Next Tab
    - `control + shift + right`
- Previous Tab
    - `control + shift + left`

### Prompt Jumps
- Previous Prompt
    - `super + shift + up`
- Next Prompt
    - `super + shift + down`
